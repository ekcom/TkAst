Help! X isn't working!

Troubleshooting steps:

  1. Make sure that X is enabled.

    Ensure that you have enabled the feature that you want on the Options page.
    You will see the checkbox next to the option checked if your current preset (whether predefined or custom) has that feature enabled.

  2. Refresh the page.

    After a feature is toggled, it will only become active on the next page load matching the URL which it runs on.

  3. Try switching presets and switching back.

    On the Options page, try switching to a different preset (like All Off) and switching back to your current preset.
    Sometimes setting up the features fails, especially after updates.

  4. Make sure that you are up to date.

    Most browsers will do automatic updates of your extensions, but you may need to fully close and reopen your browser for those updates to be applied.

  5. Try clearing your cache and cookies.

    But you knew that already ☺

  6. Escalate.

    Send me an email or submit feedback, including the following:

    • A log file from the bottom of the Options page
    • The current version of TDX that you are accessing (this should be indicated in the title of the Desktop page; it looks something like 11.4
      • )
    • Steps to reproduce the issue
